node version v21.7.0
nvm version 0.39.7
Использованная БД - MySQL.
Язык программирования JavaScript (Хотя для работы с базами данных обычно использую TypeScript)



Файл конфигурации необходимо заполнить для подключения к базе данных. (файл .env)

DB_USER - имя пользователя для подключения к бд
DB_PASSWD - пароль пользователя для подключения к бд
DB_NAME - имя бд для подключения
DB_HOST - хост для подключения к бд
TABLE_NAME - имя таблицы которая будет создаваться (по умолчанию users_table)



Прописать команду для инициализации.

-> npm i



Запуск приложения:

-> npm run start 1

-> npm run start 2 'full name' xxxx-xx-xx gender

-> npm run start 3

-> npm run start 4

-> npm run start 5

-> npm run start 6 
Проводится индексация колонок fullname и gender и выборка происходит по индексу.
первый вариант (без индексации): ~230 ms
второй вариант (с индексацией): ~2 ms



Структура проекта:

/DB - класс предоставляющий методы для работы с БД

/models - только класс User для работы с данными

/scripts - директория с используемыми скриптами

/index.js - входной файл приложения
